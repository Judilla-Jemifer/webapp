AJAX requesting a JSON File

AJAX stands for Asynchronous Javascript And XML. It is not a programming language. It uses a combination of XMLHttpRequest that is already built-in in the browser and used to request data. It also uses Javascript and HTML DOM to display or use data. However, this page will only focus on using AJAX to request a JSON file. To request JSON file, we can use GET or POST. GET is faster but POST is better when dealing with large data or using user input.
