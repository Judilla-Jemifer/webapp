HTML5 Tags - Video, Audio, and Canvas

Before HTML5 was created, videos could only be played using a plug-in like a flash in a browser. When HTML5 was created, it already had its own video element that makes it possible to embed and play the video in the webpage. The video below is embedded on this webpage using the video element. The file can be found in the data folder.



Similarly, audio could only be played using a browser plugin before HTML5. When HTML came, it enabled audio playing without plugin needed using the audio element.
